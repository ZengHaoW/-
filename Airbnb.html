计算房屋质量分数（即性价比）¶

采用指数法（地区0.2，房屋类型0.2，价格高0.2，评论数0.2，最短可租天数和全年可租天数均0.1）,其中价格和评论数用log收敛

def fun(x):
    s1 = place[x.neighbourhood_group]
    s2 = math.log(x.price, 10000)
    s3 = Type[x.room_type]
    if x.availability_365 > 365:
        s4 = 1
    else:
        s4 = x.availability_365 / 365
    if x.minimum_nights > 365:
        s5 = 0
    else:
        s5 = (365 - x.minimum_nights) / 365
    if x.number_of_reviews == 0:
        s6 = 0
    else:
        s6 = math.log(x.number_of_reviews, 629)
    return s2 * 0.2+ s1 / 3 * 0.2 + s3 / 3 * 0.2 + s4*0.1 + s5*0.1 + s6 * 0.2
rr = df.apply(fun,axis=1)
df.s = rr

小结¶

  1. 房屋现状与用户需求总结
    • 房屋大多分布在曼哈顿和布鲁克林两个区
    • 除了曼哈顿，其余四个区都是Private room的数量多于Entire home/apt（Shared room在哪里都即为稀少），租金自然是Entire home/apt>Private room>Shared room
    • 但曼哈顿的整体的房屋租金要显著高于另外四个区，其余的差距不大
    • 市场上shared room的需求较低，唯一的优点就是便宜，因为人一般都不愿与陌生人共享私人空间，除非资金是在匮乏。而曼哈顿的Private room的受欢迎程度最高（其余地方都是Private room低于Entire home/apt）。原因可能是由于曼哈顿的租金过于昂贵（尤其是Entire home/apt），而Airbnb的使用人群受众使得人们更加倾向于选择性价比高的Private room
    • 使用Airbnb的人群偏好最短可租天数小的，全年可租时段长的（即易租，短租），同时对room_type有一定要求，性价比高的房屋
  2. 通过用户需求，设计并计算房屋质量分数（即性价比）
    • 通过计算出高质量房屋在五大区内具体的分布（曼哈顿和布鲁克林与占据了绝大多数），筛选出五大区内各郡拥有高质量房屋最多的前3。平台在给用户在各大区内推荐房源时可以优先推荐这些郡旗下房源，便于用户更快的找到合适的房源，尤其是对于新用户
    • 筛选出来高质量房东与低质量房东，可以让平台维护高质量房东，提供佣金折扣以及其他福利待遇。也可以针对性的分析低质量房东的不足之处，改进房源，促进整个平台的发展。
